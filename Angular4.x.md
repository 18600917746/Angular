# Angular4.x 

> 创建项目

    ng new angular4.x --app-routing
    
> 创建 组件 

    ng generate component heroes --module=app
    
> 创建 服务

    ng generate service hero --module=app
    
> 创建 路由

    ng generate module app-routing --flat --module=app    

>  创建一个 类 文件

    ng generate class Hero
